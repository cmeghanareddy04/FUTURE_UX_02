# 🦷 SmileBright Dental — Appointment Booking App

A complete mobile app UI for appointment booking at BrightSmile Dental Clinic, Hyderabad. Designed with a mobile-first approach for patients to book, manage, and track dental appointments seamlessly.

## 📌 Project Overview

| Field | Details |
|---|---|
| **Project Name** | SmileBright Dental Booking App |
| **Category** | Mobile App UI / UX Design |
| **Business Type** | Local Service — Dental Clinic |
| **Location** | Hyderabad, Telangana, India |
| **Design Tool** | Figma / Adobe XD |
| **Platform** | iOS & Android (Mobile-first) |

---

## 🎯 Objective

Design a high-conversion mobile appointment booking system for a dental clinic that allows patients to:

- Browse dental services and specialists
- Select preferred date and time slots
- Book and confirm appointments instantly
- Receive WhatsApp confirmation within 30 minutes

| Primary Light | Teal Pale | `#059A8D` | Chip selected, backgrounds |

## 🔄 User Flow

```
Splash
  ├── Get Started → Onboarding (3 slides) → Sign Up → Home
  └── Sign In → Login → Home
                            │
                            ├── Quick Book (Home grid)
                            └── Services tab
                                    │
                                    └── Select Service
                                              │
                                              └── Select Specialist
                                                        │
                                                        └── Pick Date & Time
                                                                  │
                                                                  └── Confirm & Pay
                                                                            │
                                                                            └── ✅ Success
```

## ✅ Key Features

- **Pain-free booking** — under 60 seconds from open to confirmed
- **Real-time availability** — dot indicators on calendar for open slots
- **Specialist profiles** — ratings, experience, and availability upfront
- **Multiple payment options** — UPI, card, net banking, pay at clinic
- **WhatsApp confirmation** — instant booking confirmation via WhatsApp
- **Smart reminders** — automated reminders 24hrs before appointment
- **Cancellation policy** — free cancellation up to 2 hours before
